# 🎬 Kişiselleştirilmiş Dizi ve Film Öneri Sistemi

> **BLM0324 - Yazılım Mühendisliği Dönem Projesi**  
> **Takım Numarası:** 5  
> **Dönem:** Bahar 2025  

---

## 📌 Proje Amacı

Bu proje, kullanıcıların film ve dizi izleme alışkanlıklarını analiz ederek **kişiselleştirilmiş öneriler** sunan, yapay zeka destekli bir platform geliştirmeyi amaçlar. Kullanıcının zevklerine uygun içerikler otomatik olarak önerilerek zaman tasarrufu sağlanır ve daha verimli bir izleme deneyimi oluşturulur.

Geliştirilen sistem, kullanıcı profillerini analiz ederek hem içerik bazlı hem de işbirlikçi filtreleme yöntemlerini bir arada kullanır. Böylece kullanıcıların geçmişte beğendiği içerikler ve benzer profildeki kullanıcıların tercihleriyle desteklenen öneriler sunulabilir.

---

## 🎯 Proje Hedefleri

- 🎯 Kullanıcının ilgi alanlarına göre doğru içerikleri önermek
- 🤖 İçerik bazlı ve işbirlikçi filtreleme yöntemlerini hibrit biçimde kullanmak
- 💬 Yorum yapabilme ve sosyal etkileşim ile kullanıcı bağlılığını artırmak
- 🔍 Arama çubuğunda otomatik tamamlama ile erişilebilirliği artırmak
- 📈 Gerçek zamanlı verilerle dinamik öneri sistemini sürekli güncel tutmak
- 🧠 Makine öğrenimi tabanlı karar mekanizmalarını uzun vadede sisteme entegre etmek

## 🛠️ Kullanılan Teknolojiler

- **Frontend**: React.js
- **Backend**: Node.js, Express.js 
- **Veritabanı**: MySQL 
- **API & Veri Kaynakları**: TMDb API (içerik bilgisi çekimi)
- **Yapay Zeka**: İçerik bazlı filtreleme, işbirlikçi filtreleme, hibrit model

## 📥 Kurulum Adımları

Kuruluma başlamadan önce `.env` dosyasına API anahtarlarınızı ve veritabanı yapılandırmanızı eklemeyi unutmayınız.

### Backend Kurulumu

```bash
cd backend
npm install
npm start
```

### Frontend Kurulumu

```bash
cd frontend
npm install
npm start
```
